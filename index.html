Pencil.Gaming

An open-source, cross-platform gaming library for C# with bindings for OpenGL, OpenAL and GLFW.

Download this project as a .zip file Download this project as a tar.gz file

Pencil.Gaming is a gaming library for C#, with solid portable OpenGL support imported from OpenTK, nifty math utilities (vectors, matrices ans finally a Tau constant!), most of which are also imported from OpenTK, and solid bindings to OpenAL and GLFW, to ensure maximum cross-platform capabilities. Pencil.Gaming serves as an alternative to other popular gaming libraries, such as Microsoft XNA/MonoGame, OpenTK and SharpDX. So why use Pencil.Gaming?

Aside from binding these native libraries, Pencil.Gaming also has a few utilities, allowing you to easily

  • Load WAV files into an OpenAL buffer
  • Load images into an OpenGL texture
  • Load wavefront models into an OpenGL VBO
